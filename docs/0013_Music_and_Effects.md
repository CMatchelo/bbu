## New feature

1. Install Howler.js library

2. Inside public/audio, there are folders to each context that audios will be used

3. Side menu
- To every click in side menu, use public\audio\effects\click01.wav

4. Menu
- While the user is in main menu, use the musics in folder public\audio\menu in loop

5. In game
- While the user is in game screen, use the audios in folder public\audio\inGame in loop
- Use a fade-in fade-out effect to transition between audios so that the user wont realize the file changed
- DO NOT use the file inGame01_sneakersRubber

6. In configurations, the Audio on/off selector should now work
- If off, do not play any music or audio effect
- If on, play as normal
